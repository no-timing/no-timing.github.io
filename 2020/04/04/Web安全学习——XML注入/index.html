XML简介

XML：可扩展标记语言。XML被设计用来是传输和存储数据，XML是一种“元标记”语言，开发者可以根据自己的需要创建标记的名称。

XML结构

XML是一种树结构，从“根部”开始，然后扩展到“枝叶”，XML文档必须有根元素。

XML语法规则

  • XML 文档必须有一个根元素
  • XML元素都必须有一个关闭标签
  • XML 标签对大小写敏感
  • XML 元素必须被正确的嵌套
  • XML 属性值必须加引号
